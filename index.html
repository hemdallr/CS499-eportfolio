Category Two: Algorithms and Data Structures

Enhancement: CS 210 Item Tracker use hash table for optimization and improve search.

In this artifact it is an Item Tracker that allows the user to search for an item and display the items frequency, print a list of all items and their frequencies, and print a histogram with the item frequency. The original was enhanced by using a hash-table based dictionary to improve the average lookup time compared to the originals map structure. It takes the average lookup time from O(log n) to O(1) with the enhancement.
